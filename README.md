## Project Description
This project is an Android-based Sudoku game developed using Kotlin.  
It applies core Data Structures and Algorithms concepts such as 2D arrays,
backtracking, and recursion to generate and solve Sudoku puzzles.

## Features
- Challenge Yourself mode
- Multiple difficulty levels
- Real-time input validation
- Sudoku solver using backtracking
- Clean and user-friendly UI

## DSA Concepts Used
- 2D Arrays
- Backtracking Algorithm
- Recursion
- Searching and Traversal

## Technology Stack
- Kotlin
- Android Studio
- XML (UI Design)
